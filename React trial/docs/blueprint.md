# **App Name**: Quality Insights Chatbot

## Core Features:

- Data Ingestion & Processing: Ingest and process inspection data, purchase order (PO) details, and factory information to build a knowledge base.
- Chat Interface: Provide a user-friendly chat interface with selectable options based on the question tree.
- Guided Question Answering: Enable the chatbot to provide accurate responses based on a static dataset and a question tree, using selectable options for user input. The chatbot uses reasoning to decide when to present data as a table or plot data as a graph.
- PO Status Query: Allow users to query the status of a production order by selecting the PO number from available options.
- Inspection Details Retrieval: Enable users to retrieve specific inspection details (Inward, In-process, Final) by specifying the factory, section/building, item code, and PO number/Lot No via selectable options.
- Parameter Analysis: Provide inspection parameter analysis, allowing users to view inspection parameter results, averages, run charts, min/max readings, and operator details for a specified duration, all selectable.
- Inspection Parameter Distribution: Allow users to view the distribution of captured inspection parameters for RM/In-process/Final inspection results, filtering by item code and parameter through selectable options.

## Style Guidelines:

- Primary color: Navy blue (#2E5A88) to evoke trust and reliability, as this is a tool dealing with data.
- Background color: Light gray (#F0F4F8) to provide a clean and professional backdrop, so as not to distract from the data.
- Accent color: Teal (#3E9A8A) for interactive elements and key information highlights, providing a sense of calm insight.
- Body and headline font: 'Inter', a sans-serif font for a modern, objective, and neutral look.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use simple, clear icons to represent different data types and actions, ensuring clarity and ease of use.
- Maintain a clean and structured layout with a clear separation between the chat interface and data display areas.
